Twatcher

突然死したときに見られたくないデータを消そう

Download this project as a .zip file Download this project as a tar.gz file

目的

自分が万が一の自体に陥ってしまった時に、Twitterや秘蔵ファイルといった家族に見られたくないもの等を自動的に削除しよう。

原理

Twitterアカウントを設定しておきます。 一定期間以上Twitterに更新がなかった場合、何かあったと見なして設定してある任意のスクリプトを実行します。 設定しておけばツイートの削除や、プロフィールを更新してフォロワーさんにお知らせをすることも可能です。

使い方

  • リリースページ から最新版の twatcher-(version).zip をダウンロードします。ページ上部のダウンロードボタンはソースコードのダウンロードなので注意。
  • zipを解凍します。 Windowsなら setting.bat をダブルクリックします。 Linuxなら setting を実行します。 Macは手元にないので今のところ対応してません！ごめんなさい！
  • ブラウザが立ち上がるので、画面に従って設定やTwitterアカウントを登録してください。
  • 常に起動しているパソコンであれば、 server.bat(Windows) server(Unix) を起動しておいてください。
  • 電源を落とすことがあるパソコンであれば、 batch.bat(Windows) batch(Unix) をスタートアップで実行できるようにしてください。家族がパソコンを起動した時などに実行してくれます。

そのうちもうちょっとわかりやすく書くつもり。

ファイル解説

  • bin/ 実行スクリプトが入ってます。普通はいじる必要ありません。
  • conf/ ソフトの設定ファイルが入ってます。基本的にいじる必要ありません。
  • data/ ユーザーが設定したデータが入ります。ソフトのバージョン変更などがあった場合、このフォルダを新しいソフトに突っ込んでください。
  • lib/ 実行に必要なjarファイルが入ってます。触らないでください。
  • setting.bat(Windows) setting(Unix) 設定を変えたい場合はこれを起動してください。
  • batch.bat(Windows) batch(Unix) Twitterをチェックし、必要あらばアクションを行います。一度チェックしたら自動的にソフトが終了します。
  • server.bat(Windows) server(Unix) 1時間に一度Twitterをチェックします。ブラウザから設定も変えられます。アクションが実行されたら自動的にソフトは終了します。

凝った使い方

「お前が用意したTwitterクライアントなんか使えるか！俺は部屋に戻る！」

ごもっともです。自前のTwitterクライアントを使いたい場合はconf/application.confを開いて、 twatcher.twitter.consumer.key と twatcher.twitter.consumer.secret をご自身で用意したものに変えてください。 宣伝ツイートや勝手に作者をフォローするなどの余計な機能は入れてないので、 ツイート削除などの書き込みアクションをしないのであれば、読み権限だけで十分です。ツイート削除をしたい場合は読み書き権限にしてください。

自分でビルドしたい

sbt 0.13.8

使用にあたって

  • 旅行などでTwitterを更新してない場合でも、設定期間が経てば実行されます。それによって大事なファイルが消えても作者は責任を取りません。
  • 何かしらのバグでファイルを削除できなかったらごめんね。
  • ファイルを削除するスクリプトは近々用意する予定
